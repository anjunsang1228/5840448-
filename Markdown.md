피보나치 수열 수행시간 비교
1. 과제 개요
본 과제에서는 피보나치 수열을 두 가지 방법으로 계산하였다.
첫 번째는 반복문을 사용하는 순환적 방법이고, 두 번째는 재귀 함수를 사용하는 재귀적 방법이다.
각 방법에 대해 정수 `N`을 1부터 40까지 증가시키며 피보나치 값을 계산하고, 수행시간을 측정하여 비교하였다.
피보나치 수열은 다음과 같이 정의하였다.
`F(1) = 1`
`F(2) = 1`
`F(n) = F(n-1) + F(n-2)` (`n >= 3`)
---
2. 구현 방법
2.1 반복문 방식
반복문 방식은 앞의 두 항을 저장해 두고 다음 항을 차례대로 계산하는 방법이다.
이 방법은 한 번 계산한 값을 다시 사용하므로 불필요한 중복 계산이 없다.
따라서 수행 속도가 빠르고 효율적이다.
2.2 재귀 방식
재귀 방식은 피보나치 수열의 정의를 함수로 그대로 표현한 방법이다.
즉, `fib(n) = fib(n-1) + fib(n-2)` 형태로 구현한다.
하지만 같은 값을 여러 번 다시 계산하게 되므로 `N`이 커질수록 수행시간이 급격하게 증가한다.

---
5. 결과 분석
반복문 방식은 한 번의 반복문 안에서 이전 두 값을 이용해 다음 값을 계산하므로 시간 복잡도가 비교적 작다.
그래서 `N`이 40까지 증가해도 측정 시간이 거의 0에 가까운 수준으로 나타났다.
반면 재귀 방식은 같은 피보나치 값을 여러 번 중복해서 계산한다.
예를 들어 `fib(40)`을 구할 때 `fib(39)`와 `fib(38)`을 호출하고, 그 안에서도 다시 같은 값들을 반복 호출한다.
이러한 중복 계산 때문에 `N`이 커질수록 수행시간이 급격히 증가하였다.
실행 결과에서도 `N=30` 이후부터 재귀 방식의 수행시간이 눈에 띄게 증가하였고,
`N=40`에서는 `0.430000`까지 증가하였다.
이를 통해 반복문 방식이 재귀 방식보다 훨씬 효율적이라는 것을 확인할 수 있다.
---
6. 결론
이번 과제를 통해 피보나치 수열은 같은 문제라도 구현 방법에 따라 수행시간 차이가 크게 발생할 수 있음을 확인하였다.
반복문 방식은 빠르고 효율적이었으며, 재귀 방식은 구현은 간단하지만 중복 계산이 많아 비효율적이었다.
따라서 피보나치 수열처럼 동일한 부분 문제가 반복해서 등장하는 경우에는
단순 재귀 방식보다 반복문 방식이나 메모이제이션을 사용하는 방법이 더 적절하다고 볼 수 있다.
